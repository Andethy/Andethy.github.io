Ostinato

I've created a program that intelligently generates royalty-free music, focusing on ostinatos (short music phrases), for aspiring musicians. This project was inspired by the challenge of finding quality, royalty-free backing tracks. We used AI to generate and process musical data, confronting technical hurdles along the way. Our aim was to balance uniqueness and creativity with technical ease for artists. The project, built with various technologies including Python, React, and Google Cloud, aims for further expansion and personalization. For more details, visit the Ostinato project on Devpost.

SeniorQuote

I've developed an encryption algorithm specifically for encoding a senior quote. This project includes a Unix executable named 'Quote-D&E' which can be run in the terminal. The user can choose to encode or decode a message by passing 'e' for encoding or 'd' for decoding as an argument in the sys.argv[1]. This tool is designed for Mac systems and is written in Python.
